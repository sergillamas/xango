Editor de codi

L'editor de codi més popular actualment s'anomena VSCode.

Els editors de codi, a primera vista, són semblants a un editor de text qualsevol, com el Notepad, el Google Docs, el Word o d'altres... però amb algunes diferències notables!

  • Els editors de codi tenen syntax highlighting (colorejat de sintaxi). Básicament, això vol dir que les paraules i codi que escribim es resalten de diferents colors en base al seu significat. Els números tenen un color, les paraules reservades o clau del llenguatge de programació en tenen un altre, els operadors en tenen un altre... No cal que entenguis que volen dir aquestes paraules ara mateix, només que els diferents colors són útils per identificar l'estructura del codi que escrivim.
  • Els editors de codi normalment incorporen eines adicionals, con un navegador d'arxius per poder cambiar ràpidament entre els fitxers del nostre codi, i terminals per executar els programes. Les terminals s'explicaran més endavant.
