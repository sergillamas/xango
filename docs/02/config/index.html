Configurant l'entorn de programació

Per tornar a la pàgina principal, fes clic aquí.

Per començar a crear programes, primer calen dues coses:

  • Un compilador de codi.
  • Un programa per editar text i codi.

Compilador de codi

Un compilador de codi és un programa que tradueix els programes que hem escrit en el nostre llenguatge de programació preferit a codi màquina. Els ordinadors, internament, fan servir un codi màquina que els programadors gairebé mai escrivim directament. Per això utilitzem llenguatges d'alt nivell en el seu lloc.

El llenguatge d'alt nivell que farem servir nosaltres s'anomena Golang.

Per descarregar el compilador, has de visitar https://go.dev/dl/ i descarregar la versió per a Microsoft Windows (Windows >=10, 64-bit).

Un cop descarregat, has d'instal·lar el programa. Fes clic al fitxer que has descarregat i segueix les instruccions.

Provant si Golang funciona

Un cop ho tinguis, crea una carpeta anomenada "golang" en algun lloc accessible de l'ordinador. Dins, crea una segona carpeta anomenada "hello".

Ara, seguint el que vas aprendre de les terminals, obre una terminal dins la carpeta "hello" i executa l'ordre go version.

Si ho has instal·lat tot correctament, hauries de veure un resultat com aquest a la terminal: go version go1.23.3 windows/amd64 (pot no ser exactament el mateix resultat, però hauria de ser una cosa similar).

Molt bé! Ara executa una nova ordre: go mod init hello. Si tot ha funcionat, ara hi hauria d'haver un fitxer nou a la carpeta anomenat go.mod. És possible que des de l'explorador d'arxius no vegis l'extensió .mod del fitxer, però si fas servir l'ordre dir a la terminal, hauries de veure el nom complet.

Editor de codi

Ok, ara ja tenim el compilador, i només ens falta un editor de codi.

L'editor de codi més popular actualment s'anomena VSCode.

Els editors de codi, a primera vista, són semblants a un editor de text qualsevol, com el Notepad, el Google Docs, el Word o d'altres... però amb algunes diferències notables!

  • Els editors de codi tenen syntax highlighting (colorejat de sintaxi). Bàsicament, això vol dir que les paraules i codi que escrivim es resalten de diferents colors en base al seu significat. Els nombres tenen un color, les paraules reservades o clau del llenguatge de programació en tenen un altre, els operadors en tenen un altre... No cal que entenguis què volen dir aquestes paraules ara mateix, només que els diferents colors són útils per identificar l'estructura del codi que escrivim.
  • Els editors de codi normalment incorporen eines adicionals, con un navegador d'arxius per poder cambiar ràpidament entre els fitxers del nostre codi, buscadors de text i terminals per executar ordres.

El primer pas és descarregar VSCode. Fes clic aquí i busca l'opció de descarregar per Windows. Igual que amb el compilador per a Golang, busca el fitxer que has descarregat i instal·la'l.

Familiaritzant-te amb VSCode

En una terminal oberta a la carpeta "hello", executa l'ordre code %cd%. Si tot va bé, se t'hauria d'obrir VSCode, i hauries de veure que el fitxer go.mod que hem creat abans apareix a l'explorador de fitxers de l'esquerra.

Si per algun motiu tanques accidentalment l'explorador de fitxers de VSCode, el pots tornar a fer aparèixer amb la combinació de tecles "CTRL + SHIFT + E" (el SHIFT és la fletxa que apunta cap amunt, just a sobre de la tecla CTRL).

Ara, fes clic a la icona per crear un nou fitxer a la part superior de l'explorador de fitxers de VSCode, i anomena'l main.go

Si fas doble click sobre el fitxer, se t'hauria d'obrir a la finestra principal de VSCode.

Ara, copia el següent contigut dins el fitxer main.go:

package main

import "fmt"

func main() {
	fmt.Print("Hello World\n")
}

Finalment, des de la terminal que tens oberta a la carpeta "hello", dins la qual tens tots aquests fitxers, executa go run main.go

Acabes d'executar el teu primer programa!
